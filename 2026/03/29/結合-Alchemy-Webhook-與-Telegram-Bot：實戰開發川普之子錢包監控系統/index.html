2026-03-29

一、前言：為什麼要追蹤大戶錢包？

川普談關稅時我在當兵 白天根本不能用手機(火力連真滴可悲) 台股跌停三天是跑都跑不掉 從那時我理解到 現在的k線根本是川普在畫的 技術分析還是有其風險 美國只要有個風吹草動 不管是台股還是加密貨幣馬上就有波動 加密貨幣更是明顯 既然這樣 那打不過就加入 與其在那邊SMC 不如追蹤勝率80以上的大老 沒錯 就是川普他兒子 傳聞川普再公布消息前 幕僚跟他兒子會大量買進 那我也要跟

這篇分享如何利用 Python FastAPI、Alchemy Webhook 與 Telegram API，開發一套鏈上監控系統，實現「大戶一動，手機秒通」的自動化監控。

二、技術架構

採用 Push (推送) 機制而非傳統的輪詢（Polling），確保最低延遲：

  1. Alchemy Notify: 監控 Ethereum 鏈上特定地址（Target: 0xc2a3...e5f2）。
  2. FastAPI 後端: 接收來自 Alchemy 的 Webhook POST 請求。
  3. Ngrok 隧道: 在開發環境中將本地服務曝露至公網。
  4. Telegram Bot API: 將格式化後的交易資訊即時推送到手機。
