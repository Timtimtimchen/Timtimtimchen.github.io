前言

最近老闆開會強調效率這件事,於是找到這款工作流程開源軟體,相較於Flowise,N8N不止提供AI,還支持多種節點,可以讓各部門解決行政上重複問題,加快工作速度

介紹

n8n 是一個開源而且免費的自動化工作流平台，它提供了簡單易用的 Web UI，拉出需要節點串成流程圖，不用寫代碼就能夠簡單的創建自己的工作流。
